# Josh Bot 2.0 - React Front-end/Admin Panel

### To start:
###### Navigate into the ui directory:
```bash
$ cd josh_bot_2.0/ui
```
###### Download the dependencies:
```bash
$ npm install
```
###### Run the start script:
```bash
$ npm start
```

*The app will be available in the browser at http://localhost:3000*
