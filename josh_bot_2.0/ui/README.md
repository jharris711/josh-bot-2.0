# Josh Bot 2.0 - The Front-End

### To start the UI:
###### Navigate into the ui directory:
```bash
$ cd josh_bot_2.0/ui
```

###### Run the start command:
```bash
$ npm start
```

*UI will be available in the browser at: http://localhost:3000*
